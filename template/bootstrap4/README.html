README - Bootstrap4 Template for Opac SLiMS Akasia 8.x.x
How to Install
  • Ekstrak file zip yang sudah diunduh
  • Copy folder hasil ekstrak tadi, lalu paste ke folder template Opac Akasia
  • Masuk ke halaman Login Pustakawan (Librarian Login)
  • Untuk Akasia 8.3, klik menu System lalu pilih tampilan template pengguna dengan template ini (Bootstrap4)
  • Untuk Akasia 8.3.1, klik menu System > Theme kemudian aktifkan template Bootstrap4
  • Selesai

Customization
Mengganti Logo

Logo pada template berformat *.png dengan ukuran 60 x 60 pixel. Logo ditampilkan di tiga area, yaitu area navigasi (partials/nav.php), area sidebar promo (partials/sidebar/promo.php), dan area Login Admin (login_template.inc.php)

Logo pada Navigasi

Logo pada sidebar Promo

Logo pada Login Admin area

Untuk mengganti logo, Anda bisa menggunakan Adobe Photoshop atau CorelDRAW. Sesuaikan dengan ukuran logo tersebut, yaitu 60 x 60 pixel.
Jika menggunakan Photoshop, buat gackground-nya transparan, lalu Save As dengan format png (*.png). Dan jika menggunakan CorelDRAW, export objek, lalu pilih format png, dan aktifkan centang background transparency.

Menonaktifkan/Merubah Bagian-bagian Template
  • Fixed Social Menu (partials/social-sidemenu.php). Untuk menonaktifkannya, buka file index_template.inc.php yang ada pada folder template Bootstrap4 ini dengan text editor (misalnya Notepad atau Notepad+), cari kode
    include "partials/social-sidemenu.php";
    lalu blok dengan menggunakan dua garis miring // di depannya, sehingga menjadi
    //include "partials/social-sidemenu.php";

    Perhatikan contoh di bawah ini:
    
<?php 
// Header and Menu
include "partials/nav.php";

// social side-menu
//include "partials/social-sideMenu.php";
?>
    Demikian juga untuk komponen lainnya, misalnya Sidebar Main Menu (partials/sidebar-sideMenu.php), atau Sidebar Promo (partials/sidebar-promo.php). Untuk menonaktifkannya, blok dengan menggunakan dua garis miring // di depannya.

    Perhatikan contoh code yang berwarna merah di bawah ini:
    
<div class="col-xl-4 col-lg-4 col-md-4 col-sm-12 animated fadeInUp">
	<div class="sidebar animated fadeInUp">
		<?php 
		// sidebar search result information
		include "partials/sidebar/searchResult.php";
		// sidebar information
		include "partials/sidebar/info.php";
		// sidebar main menu
		//include "partials/sidebar/sideMenu.php";
		// sidebar promo
		//include "partials/sidebar/promo.php";
		// search cluster
		include "partials/sidebar/searchCluster.php";
		?>
	</div>
</div>
  • Detail Record (detail_template.php). Pada bagian detail record (detail_template.php), diganti menjadi model nav-tab, jika tidak cocok dengan model ini, telah disediakan model default-nya.
    Caranya, pertama-tama ganti nama (rename) file detail_template.php menjadi detail_template_nav-tab.php. Berikutnya, ganti nama file detail_template_default.php menjadi detail_template.php
  • Mengganti navigasi utama. Ada dua jenis navigasi utama yang disediakan, yaitu yang menggunakan logo full image (*.png), dan yang menggunakan logo image dan text. Navigasi dengan full logo image ada pada file yang bernama nav-v2.php (partials.nav-v2.php).
    Untuk menggantinya, buka index_template.inc.php, dan cari
    <?php include "partials/nav.php";?>
    ganti menjadi
    <?php include "partials/nav-v2.php";?>
